premiumize-explorer

A two-pane desktop file browser for Premiumize.me cloud storage,
built with C++20 and Qt6.

View on GitHub
API & Architecture docs

Features

Two-pane layout Total Commander-style — local filesystem on the left, Premiumize.me cloud on the right.
Drag & drop transfers Upload by dragging from the local pane, download by dragging to it.
Transfer progress window Non-blocking window showing speed, ETA, elapsed time, and bytes per file. Auto-scrolls to the active transfer; stays visible when the main window is minimised; optional Stay on top (X11).
Folder navigation Breadcrumb path label and an always-visible ↑ Up entry at the top of each pane.
Cloud operations Create folders, delete files or folders, and a right-click context menu. Multi-select with Ctrl+Click / Shift+Click — download multiple files and folders in one action; folders download as .zip archives.
Cloud transfers monitor Live view of server-side Premiumize transfers (torrents, URL downloads) with status, progress, speed and ETA. Auto-refreshes every 5 s.
Batch download wizard Search cloud files by keyword, select with checkboxes, pick a destination, and download all with dual progress bars, a live timer, and a per-file status list (pending / active / done / error). File → Batch Download…
API log window Timestamped log of every HTTP request and response with timing. Saveable to file.
Dark mode Qlementine dark theme — toggle via View → Dark Mode and persisted across restarts.
Persistent settings API key, last local path, window geometry, and splitter position saved automatically.

Build

Requires CMake 3.21+, Qt 6.5+ (Core, Widgets, Network, Svg), and a C++20 compiler. Qlementine is fetched automatically. 

# First-time configure (~30 s on a fresh clone)
cmake -B build/debug -DCMAKE_BUILD_TYPE=Debug

# Build
cmake --build build/debug --parallel

# Run
./build/debug/src/premiumize-explorer
# Release build
cmake -B build/release -DCMAKE_BUILD_TYPE=Release
cmake --build build/release --parallel

First launch

On first run you will be prompted for your Premiumize.me API key. Find it at premiumize.me/account under the API section. The key is saved immediately — you won't be asked again.

To change the key later: File → Change API Key…

Download files / foldersDrag from the cloud pane onto the local pane; or select items (Ctrl+Click / Shift+Click) and right-click → Download — files download directly, folders as .zip

View cloud transfers (server-side)View → Cloud Transfers — shows torrents and URL downloads Premiumize is processing for you
Refresh the cloud viewClick ⟳ in the cloud toolbar
Batch download filesFile → Batch Download… Ctrl+Shift+D — search by keyword, check the files you want, pick a destination, then download them all
View API logView → API Log — timestamped request/response log; use Save to File… to export
Toggle dark modeView → Dark Mode
